# Blog_mDB
This is a blogging website build using Node.JS and  MongoDB  with Markdown syntax support .

## Technologies used :
1. Bootstrap  -- Frontend
2. javascript -- programing language
3. Node.js    -- Platform
4. ejs        -- Template
5. express    -- Framwork
6. mongoose   -- ODM
7. MongoDB    -- Dtabase
8. Atlas      -- Cloud data store
9. Heroku     -- Deployment 

## Installation :
```$npm install```

## Run :
```npm start```
